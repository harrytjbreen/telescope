# Electronics compiled notes
## Motor Drivers: DRV8825 
- specs
  - datasheet drv8825.pdf
  - 8.2V - 45V out, 1A (cont) 2.5A (max) current
  - 1/32 min step
- notes
  - vulnerable to voltage spikes, need 47muF+ capacitor across VMOT and GND
  - https://lastminuteengineers.com/drv8825-stepper-motor-driver-arduino-tutorial/
  - install heatsink for actual 2.5A usage 

  
- Motors: NEMA17
